# MyReads: EgFWD React cross-skilling ND

---

#### Project Stack:

Client side: React with _TypeScript_

Server side: Express / MongoDB

Styling: Tailwind CSS

---

#### Installation

kindly run `npm i` for both folders (_client_ and _server_)

---

#### Start the app

###### Client side:

    cd client
    npm start

###### Server side:

    cd server
    npm start
